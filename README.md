# 📱 Counter App

A simple Android Counter application built using modern Android development practices.

## Features

- Increment Counter
- Decrement Counter (minimum 0)
- Persistent storage using Room Database
- MVVM Architecture
- Jetpack Compose UI
- StateFlow
- Dagger Hilt

## Tech Stack

- Kotlin
- Jetpack Compose
- Room Database
- Flow
- MVVM
- Hilt

## Architecture

```text
UI
↓
ViewModel
↓
Repository
↓
Room Database
```

## Installation

Clone repository:

```bash
git clone https://github.com/amitbehera96/Counter.git
```

Open in Android Studio and Run.
